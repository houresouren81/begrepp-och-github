Begrepp och Github Uppgift

Githubflow

Med Github och git kan man dela med andra sin del av project eller hel project
Det kan man ändra eller uppdetera eller utveckla eller
lägga till andras project by clona andras koder och pusha dem till main project.

Flexbox

Det är modul ,man kan fixa blockes, knappar, rader, listor och meny.
Man kan kombinera mellan img ,text ochså , och göra ordning i style av hemsidan.
och sen man kan använda med script ex file koppla functions till de knapparna eller menum som man fixat med flexbox. Det finns olicka sort av flexbox container resposiv och och.

DOM(Document Object Model)

Det är betyder struktur och uppbyggnads av html och css och hur vi jobbar med det.
Det är som karta lägga en object och när man hittar DOM trädet ,
man kan skriva koder i javascript och ändra innhalet av html.

Finkod

Det betyder att man måste skriva tydligt koder ,
för det ska vara lätt att läsa det. ibland man behöver uppdetera nåt efter några år månader,
eller nån annan ska jobba på det som vi har gjort med simsällskåp kod inspect. eller ibland jätte stor project behöver dela koderna till små o lätta koder för det ska funka rätt.

Ramverk

Det är som tilllägg tjänst som kodbibliotek , som nån har skrivit methoder o koder i det.
och till oss vi passar på att använda de färdiga koderna för spara massa tid till oss. tyvärr man kan inte ändra nåt kod i dem.
